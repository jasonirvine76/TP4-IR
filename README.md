## About Project (English)
This Search Engine was developed using the **MS MARCO** dataset, a large-scale collection curated by Microsoft featuring real-world queries from the Bing search engine and their corresponding relevant documents.

Technical highlights of this project include:
* **Dataset Management**: Processed **1,000,000** documents, **1,185,869** qrels (query-document relevance pairs), and **76,000** queries.
* **Ranking Model**: Developed a **Learning to Rank (LETOR)** model trained on a subset of **200,000** documents to optimize search relevance.
* **Core Engineering**: Implemented an **inverted index** and utilized **Multi-way Merge Sort** to manage large-scale indexing on disk efficiently.

## About Project (Indonesia)

Search Engine dengan MSMARCRO dataset. MSMARCRO merupakan data yang dikumpulkan oleh microsoft berdasarkan pertanyaan yang dalam mensin pencari BING dan dokumen terkait pertanyaan tersebut.  Terdapat 1.000.000 document, 1.185.869 qrels, dan 76.000 query yang diambil dari MSMARCRO.  Sebanyak 200.000 document digunakan untuk membuat model LETOR.
